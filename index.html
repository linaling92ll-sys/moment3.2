Vad har jag gjort för att lösa uppgiften?

För att lösa uppgiften så sökte jag på internet och hittade väldigt bra tips på w3schools/How TO-Fullscreen Video.
Jag la in två olika videoformat för att försäkra mig om att alla kan se videon. Jag hittade information angående detta också på w3schools/HTML Video.

Jag började med att lägga in själva videon som jag ville ha som bakgrund och så la jag den i olika videoformat.
Sedan la jag en div runt själva den här texten för att jag skulle kunna styla den med hjälp av CSS.

Jag stötte på några problem när jag skulle skriva CSSen för den här webbplatsen.
Jag fick en horisontellscroll och för att få bort den så hittade jag att man kan använda overflow-x i css.
Det denna gör är att man kan bestämma om innehållet ska "klippas bort", lägga till en scroller. Här är länken till sidan som jag hittade det på Overflow.

Jag hade problem med att knapparna på videon inte syntes så då sökte jag lite och hittade information om Z-index.
Den här bestämmer ordningen som innehållet ska placeras på. Så att de inte hamnar bakom något annat element.
